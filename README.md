#  Bank Management System (C++ OOP)

##  Description
This is a simple Bank Management System built using C++.
The project demonstrates Object-Oriented Programming concepts.

##  Features
- Add Clients
- Add Accounts (Savings / Current)
- Deposit
- Withdraw
- Apply Interest
- Overdraft Support
- Show Client Accounts

##  Concepts Used
- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
- Vectors

##  How to Run
1. Open the project in any C++ compiler
2. Compile main.cpp
3. Run the program
